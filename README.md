# Purple Team Attack & Detection Lab

<h2> Goal of the Project</h2>
Simulate a cyberattack and detect it using a SIEM. 
<br>
In this project:
<br>
  1. Launch Attack <br>
  2. Collect Logs <br>
  3. Detect suspicious activity <br>
  4. Investigate the attack <br>
