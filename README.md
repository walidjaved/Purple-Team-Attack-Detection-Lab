# Purple Team Attack & Detection Lab

<h2> Goal of the Project</h2>
Simulate a cyberattack and detect it using a SIEM. 
<br>
In this project:
<ol>
  <li>Launch Attack</li>
  <li>Collect Logs</li>
  <li>Detect suspicious activity</li>
  <li>Investigate the attack</li>
</ol>
